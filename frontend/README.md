# Frontend (React + Tailwind + Jest)

## Run

```bash
cd frontend
npm install
npm run dev
```

Default app URL: `http://localhost:5173`

## Test

```bash
cd frontend
npm test
```

## Authentication Features Implemented

- Login form (admin/recruiter)
- First admin bootstrap form
- Authenticated control panel
- Admin-only recruiter creation UI
- SweetAlert2 notifications
- Loading skeleton + spinner states
- Responsive and animated neon UI
- Role-aware rendering

## Env Option

Create `.env` if needed:

```bash
VITE_API_BASE_URL=http://localhost:5000
```
